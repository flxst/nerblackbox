    run_experiment(self, experiment_name, from_config=False, model=None, dataset=None, from_preset='adaptive', **kwargs_optional)

    run a single experiment.

    Note:

    • from_config == True -> experiment config file is used, no other optional arguments will be used

    • from_config == False -> experiment config file is created dynamically, optional arguments will be used

      • model and dataset are mandatory.

      • All other arguments relate to hyperparameters and are optional. If not specified, they are taken using the following hierarchy:

        1) optional argument

        2) from_preset (adaptive, original, stable), which specifies e.g. the hyperparameters "max_epochs", "early_stopping", "lr_schedule"

        3) default experiment configuration

    Parameters:

    experiment_name str
